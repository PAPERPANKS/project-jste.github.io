    Any texts including headings, titles and paragraphs.

    Properties:

    Property Value
    name The name you have given to this element.
    container The name of the container which will contain this element.
    width The generic width of this element.
    length The generic length of this element.
    text The text you have been inserted.
    font_color The color of the text.
    background The background of this element.
    font_size The size of the text in this element.
    font_style The font style of the text in this element.
    thickness The thickness of the font of the text in this element.
    animation The animation of this element on loading it.
    transparency The transparency of this element Takes a value from 1 to 100.
    distance_from_the_top The distance between the top edge of the container contains this element and this element.
    distance_from_the_bottom The distance between the bottom edge of the container contains this element and this element.
    distance_from_the_left The distance between the left edge of the container contains this element and this element.
    distance_from_the_right The distance between the right edge of the container contains this element and this element.
    position The position state of this element Takes fixed, absolute or relative.
    commands The commands which will be executed when making a specific action o nthis element.

    Tooltip

    add a button with the following properties:
name: 'A_sample_button',

add a tooltip with the following properties:
name: 'A_sample_tooltip',
text: 'This is a sample tooltip',
font_color: 'white',
emitter: 'A_sample_button',
direction: 'from_the_bottom',
transparency: '50',
commands: 'When it has been clicked, remove it'.

    Tooltip to make it easier for you to add a description for any of your elements.

    Properties:

    Property Value
    name The name you have given to this element.
    container The name of the container which will contain this element.
    text The description of the element from which the tooltip will be emitted.
    emitter The element from which the tooltip will be emitted.
    direction The direction from which the tooltip will be emitted It may be from_the_right, from_the_left, from_the_top or from_the_bottom which is the default value.
    width The generic width of this element.
    length The generic length of this element.
    font_style The font style of the text in this element.
    font_color The color of the icon.
    background The background of this element.
    font_size The size of the icon.
    animation The animation of this element on loading it.
    transparency The transparency of this element Takes a value from 1 to 100.
    commands The commands which will be executed when making a specific action on this element.

    Events

    Most of the events in Jste English edition are started with the word When and end with , to facilitate writing and reading them and in the fowllowing you will find a list of all of these events and their descriptions:

    Event Description
    When it has been clicked, It is fired when someone clicks the element.
    When the mouse pointer has been moved over it, It is fired when someone moves the mouse pointer over the element.
    When the mouse pointer has been moved away from it, It is fired if someone was moving the mouse pointer over the element and then moved it away from it.
    When the mouse pointer is being moved out of it, It is fired when the mouse pointer is in the state of moving outside the element zone.
    When the mouse pointer is being moved over it, It is fired when the mouse pointer is in the state of moving over the element zone.
    When it is no longer clicked, It is fired if someone was clicking the element and then relased the mouse button up.
    When it has been double clicked, It is fired when someone double clicked the element.
    When it has been right clicked, It is fired when someone right clicked the element.
    When a key has been pressed while focusing it, It is fired when someone pressed a keyboard key while focusing the element Mostly used for textboxes.
    When a key is being pressed while focusing it, It is fired when someone is pressing a keyboard key while focusing the element Mostly used for textboxes.
    When its contents have been modified, It is fired when the value of the element has been changed e.g. The text in the textbox.
    When it has been focused, It is fired when the element is focused Mostly used for textboxes.
    When it is being focused, It is fired when the element is being focused Mostly used for textboxes.
    When it is no longer focused, It is fired when the element is no longer focused Mostly used for textboxes.
    When the data of this form has been sent, It is fired when the data of the element has been sent Can be used only for forms.
    When it has been scrolled, It is fired if someone scrolled this element in any direction.
