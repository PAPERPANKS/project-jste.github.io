What's Jste ?


Jste is the first JavaScript framework which is based on plain text translated to many languages

This will make everyone in the world be able to develop his/her own app or website without any coding skills

Translated

Jste syntax has been made as a plain text translated to various languages from many countries which will make programming easier and simplified.

Fast

Any website made with Jste is stored in a single html file and all the JS libraries and stylesheets are stored locally on the user's device* which saves loading time and data usage.

Secure

Say goodbye to the sudden nudes & sexual photos and videos or the accessing of your kids to the +18 contents as every website or app made by Jste is rated and moderated depending on specific guidelines

* Jste manager is required to be installed on the user's device
